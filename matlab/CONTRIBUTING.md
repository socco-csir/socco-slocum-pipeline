## Toolbox development process
1) The first version of the toolbox was written by [Gerd Krahmann](https://git.geomar.de/gerd-krahmann) from 2017 to 2022. The flow speed dependent part of the salinity correction is based on code from Barolome Garau [0000-0001-5805-6195](https://orcid.org/0000-0001-5805-6195).

## Contributions 
[Gerd Krahmann](https://git.geomar.de/gerd-krahmann) initiated the development of the toolbox in March 2008 after acquiring 
several Deep Slocum G1 gliders manufactured by Teledyne Webb Research.
